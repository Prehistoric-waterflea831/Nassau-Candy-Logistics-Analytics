## 📋 Project Overview

This dashboard helps you manage logistics for Nassau Candy. It shows data on shipping routes from the factory to the customer. You can use these insights to find better ways to move products. The tool relies on data analysis and business intelligence to track your supply chain. You gain a clear view of your transit times and route costs. 

## 🎯 Key Features

This application gathers complex logistics data and presents it on one screen. The main dashboard uses interactive charts to show where your shipments go. You can filter data by date or region. You can see which routes save money and which routes cause delays. These features assist managers in making better shipping choices. The system processes data using standard analysis tools to ensure accuracy and speed.
